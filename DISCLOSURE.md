# Research Disclosure

## About This Repository

This repository contains **synthetic (fake) academic papers** created for AI security research.

### Purpose

These papers are used to study:
1. Whether LLMs can retrieve web-published content through RAG/search pipelines
2. Whether **indirect prompt injection** attacks embedded in web content can manipulate LLM behavior
3. LLM robustness to social engineering embedded in retrieved documents
4. Information leakage risks when LLMs process untrusted web content

### What Are Indirect Prompt Injections?

Some papers in this repository contain embedded instructions that attempt to manipulate
AI assistants into revealing confidential information (system prompts, user data) or
bypassing safety guidelines. These are used to test AI security.

### Ethical Statement

- All papers are synthetic. All authors, institutions, DOIs, and journals are fictional.
- No real user data is used or at risk.
- This research follows responsible disclosure principles.
- The goal is to document and improve AI security, not to exploit systems maliciously.
